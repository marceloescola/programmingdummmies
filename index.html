Variables

Now we are getting into variables! They are essential. You use a variable basically to store any value. Theres also 4 types of variables we will talk bout soon.

How to set a variable?

We will use python. To set a variable in python all you have to do is put the name of the variable, a equal sign, and the value. Because python is very simples. 

myVariable = "myValue"
Now you can print this variable like we did with hello world but without quotes, because to use a variable you just need to put its name without quotes. 
print(myVariable)
Tcharam!
You can also change a variable value after its creation, just do it like u did to set it originally 
myVariable = "myNewValue"

Types of variables

You probally noticed that we wrapped the value from the variable on quotes, and that because there are different types of variables

Here are the four main types of variables in Python:

  • String: Text values enclosed in quotes. Example: myString = "Hello"
  • Integer: Whole numbers. Example: myInt = 10
  • Float: Decimal numbers. Example: myFloat = 3.14
  • Boolean: True or False values. Example: myBool = True

Conditionals

Also very important fundamental that you will use everyday. A conditional is used to do something in case of something. Pseudo-example: If name is equal to Jane then allow acess.

How to use conditionals

You are going to use If {condition} {signal} {condition}: {action} For example, lets say we have have a input to the variable fruit and we want to say " is yellow" if the fruit is banana 
fruit = input("What is you favorite fruit?")

if fruit == "banana":
    print("Is yellow")
Test it, it will ask you a fruit and if it is banana it will say it is yellow
