First of all, you MUST print your first "Hello World". Else you'll be cursed in programming forever.

To do your first "Hello World", you need to install Python, which is the language we are using in this article (there are multiple tutorials for that, so we won't teach how to install).

Now open Visual Studio Code or any other code editor, and create a file called main.py. Into it you will insert

print("Hello World!")

That says to the program to say "Hello World!", now open a terminal on your pc or code editor.
